tar命令

tar命令可以把一大堆文件和目录打包成一个文件

打包和压缩

打包是指将一大堆文件或目录变成一个总的文件；压缩则是将一个大的文件通过压缩算法变成一个小文件。

打包命令格式

[root@localhost ~]# tar [选项] [-f 压缩包名] 源文件或目录

选项：

  • -c：打包；
  • -f：指定文件名，注意要正确指定扩展名；
  • -v：显式打包文件的过程

解打包命令格式

[root@localhost ~]# tar [选项] 压缩包

选项：

  • -x：解打包
  • -f：指定文件名
  • -v：显示过程
  • -t：列出文件内容
  • -C：目录，指定解打包的位置

.tar.gz 和 .tar.bz2 格式

[root@localhost ~]# tar [选项] 压缩包 源文件或目录

选项：

  • -z：压缩和解压缩成".tar.gz"格式
  • -j：压缩和解压缩成".tar.bz2"格式
